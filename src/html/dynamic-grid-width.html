Dynamic grid width

Drag or resize some widgets to the right side. Use max_cols option to set a maximum number of columns for the grid.

初始化实例对象时的 min_rows, min_cols, max_rows, max_cols 可以限制容器内最小/大 行列数量， 影响了ul的宽高内联属性的设置

存在BUG,当 avoid_overlapped_widgets属性与宽高都有限制时，宽高限制优先级更高，即当限定宽高内无法挤下对应的盒子时，将overlap(重叠格子)
